# Edge Cognitive Load Adaptive System

## Overview
This project aims to detect cognitive load using multiple sensors 
(temperature, light, motion) and adapt system behavior in real-time 
on edge devices like ESP32.

## Features
- Multi-sensor data fusion
- Real-time stress/cognitive load detection
- Adaptive alert system
- Edge-based processing (low latency)
